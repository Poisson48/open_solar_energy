Physique solaire précise

Modèles de référence académique, pas de coefficients forfaitaires.

  • Transposition HDKR anisotrope (Hay-Davies-Klucher-Reindl 1990)
  • Intégration numérique Rb (Braun & Mitchell) — tout azimut
  • Correction thermique NOCT réelle (IEC 61215)
  • Optimisation inclinaison + azimut (91×13 combinaisons)
  • VAN 25 ans, LCOE, payback actualisé (+3 %/an hausse élec)
🏠

Dimensionnement réseau

Depuis ta facture EDF ou l'export CSV de ton espace Enedis.

  • Import CSV Enedis multi-format (Wh/kWh, 30min, HP/HC)
  • 3 stratégies : ROI optimal / autoconso max / couverture cible
  • Calcul économies, surplus injecté, prime autoconsommation
  • Tarif Base ou HP/HC, abonnement déduit
🔋

Hors réseau + batterie

Simulation horaire complète avec 6 technologies de stockage.

  • Import Enedis direct (ZIP ou CSV) dans l'onglet hors réseau
  • Simulation horaire si données 30min disponibles
  • SOC mensuel porté sur 12 mois — bilan annuel réaliste
  • Matrice heatmap PV × batterie, jours de déficit
  • LFP, AGM, NMC recondit. (Leaf, Zoé, Tesla)
🌤️

Données météo

Plusieurs sources, une seule interface.

  • Open-Meteo (gratuit, CORS natif, moy. 2020–2023)
  • PVGIS JRC (via proxy ou import JSON manuel)
  • 4 villes démo embarquées (Paris, Toulouse, Nice, Bordeaux)
  • Carte Leaflet interactive — clic ou glissé pour placer le site
📄

Devis professionnel

Génération PDF A4 complète depuis l'application.

  • Infos installateur (société, SIRET, RGE) sauvegardées
  • 7 lignes de coût éditables + remise + TVA (5,5 / 10 / 20 %)
  • Import automatique des résultats du dimensionnement
  • Numéro auto-généré, bloc signature client, impression directe
💾

Gestion de projets

Plusieurs scénarios, zéro cloud.

  • Sauvegarde locale (localStorage), pas de compte requis
  • Clonage pour comparer orientations, technologies, surfaces
  • Export / import JSON — partage entre machines
  • Ctrl+S pour sauvegarder à tout moment
Installation

Prêt en 30 secondes

Aucune dépendance, aucun runtime à installer. Un seul fichier exécutable.

Linux x64 Disponible

1
Télécharger

Récupérer Open-Solar-Energy-*.AppImage depuis la page Releases.

2
Rendre exécutable
chmod +x Open-Solar-Energy-*.AppImage
3
Lancer
./Open-Solar-Energy-*.AppImage

Les mises à jour sont automatiques — l'application vérifie les nouvelles versions au démarrage et propose le redémarrage.

Windows Bientôt

1
Télécharger

Le fichier Open-Solar-Energy-*-Setup.exe sera disponible dans les releases.

2
Installer

Double-clic sur le fichier .exe et suivre l'assistant.

3
Alternative — navigateur
# Cloner le dépôt : git clone https://github.com/Poisson48/open_solar_energy.git cd open_solar_energy # Lancer : serve.bat

La version navigateur fonctionne sur tout OS avec Python 3 installé.
