Gratuit & Open Source

Open Solar Energy

Dimensionnement et simulation de systèmes photovoltaïques — 100 % local, sans serveur, sans inscription.

Télécharger pour Linux (.AppImage)
Linux x64  ·  Mises à jour automatiques  ·  Code source sur GitHub

Ce que fait l'application

☀️

Simulation solaire physique

Transposition HDKR (Hay-Davies-Klucher-Reindl), correction thermique NOCT, optimisation automatique inclinaison + azimut.

🏠

Dimensionnement réseau

Import CSV Enedis, 3 stratégies (ROI optimal, autoconsommation max, couverture %). Calcul économies, surplus, ROI, VAN 25 ans, LCOE.

🔋

Hors réseau + batterie

6 technologies batterie (LFP, AGM, NMC recondit.), simulation horaire, matrice PV × batterie, jours de déficit.

🌤️

Données météo intégrées

Import Open-Meteo (gratuit, CORS natif) ou PVGIS. Démo embarquée : Paris, Toulouse, Nice, Bordeaux. Carte interactive Leaflet.

📄

Devis professionnel

Formulaire complet installateur + client + chantier. Export PDF A4 prêt à envoyer, TVA configurable (5,5 % / 10 % / 20 %).

💾

Gestion de projets

Sauvegarde locale, clonage pour comparer scénarios, export / import JSON. Fonctionne 100 % hors connexion.

Installation en 3 étapes

1
Télécharger

Récupérer le fichier Open-Solar-Energy-*.AppImage depuis la page Releases.

2
Rendre exécutable

chmod +x Open-Solar-Energy-*.AppImage

3
Lancer

Double-clic sur le fichier ou ./Open-Solar-Energy-*.AppImage

Les mises à jour sont automatiques : l'application vérifie et installe les nouvelles versions au démarrage.

Alternative open-source à PVGIS

Aucune donnée envoyée à un serveur. Tous les calculs s'effectuent localement.
